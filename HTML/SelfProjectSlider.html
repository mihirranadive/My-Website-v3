1. Cookie Lock

    A small project done for preventing automatic sign ins to websites. Using C#/.NET when a user opens Chrome, he's prompted for a password. Chrome still opens on an incorrect password but deletes the cookies which would sign in the users into some regularly visited websites. Done mostly to ward off roommates using each others laptops.

  2. Android Hand Gesture recognition

    Started with the inspiration for reading sign language via the camera in phones, this app is developed up to the stage that it can recognize three gestures from the user which can be used to do certain phone operations. Done in Java with Eclipse with the android emulator.
